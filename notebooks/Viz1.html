After walking through the tutorial, maybe height isn’t a good predictor of kills production. What if we changed height to total attacks as our predictor of kills output?

To do this, first create a dataframe called attackers that is all players who had greater than 0 total attacks.

To be 100 percent clear – you are no longer using height. If your code has height in it, you are doing it wrong.

Copy and paste your code that creates that dataframe.

attackers <- players |> filter(total_attacks>0)

Model your new data in a linear model. Report your Adjusted R-squared value.
